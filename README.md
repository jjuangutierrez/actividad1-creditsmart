# CreditSmart

Plataforma web para simulación y solicitud de créditos financieros.

## Características

- Catálogo de productos financieros con información detallada
- Simulador de créditos con filtros de búsqueda
- Formulario de solicitud completo
- Diseño responsive para móviles, tablets y escritorio
- Interfaz moderna con tema oscuro

## Instalación

1. Clona el repositorio:

bash
git clone https://github.com/tu-usuario/creditsmart.git
cd creditsmart


2. Abre index.html en tu navegador o usa un servidor local con Live Server

# Páginas

# index.html
Página principal con hero section y catálogo de 5 productos financieros (Libre Inversión, Vehículo, Vivienda, Educativo, Empresarial).

# simulador.html
Simulador con búsqueda por nombre y filtro por rango de monto. Muestra resultados en tarjetas.

# solicitar.html
Formulario completo dividido en tres secciones:
- Datos Personales
- Datos del Crédito
- Datos Laborales
